# Finance Analysis Pack

## Overview

A toolkit for automating standard financial reporting and analysis tasks. Contains Python scripts to generate Income Statements (P&L) and perform Variance Analysis (Actual vs Budget) from raw transaction data.

## Contents

- **Income Statement Generator**: Script to structure and format a P&L statement from a list of accounts and transactions.
- **Variance Analysis Template**: Script to compare actual results against budget/forecast and highlight significant deviations.
- **Sample Data**: A CSV file with sample financial data to test the scripts.

## Use Cases

- Automating monthly financial close reporting
- Quickly identifying budget overruns across departments
- Standardizing the format of financial statements
- preparing data for executive dashboards

## Complexity

Intermediate - Requires basic understanding of financial accounting and Python (Pandas).
