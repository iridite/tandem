# Expected Outputs

## Console Output

The scripts print formatted tables to the console for quick verification:

```text
Income Statement (2023-Q1)
----------------------------------------
Revenue                      $500,000.00
  Product Sales              $380,000.00
  Service Revenue            $120,000.00
Cost of Goods Sold          ($200,000.00)
----------------------------------------
Gross Profit                 $300,000.00
...
```

## Files (Optional)

If you uncomment the export lines, the scripts generate:

- `income_statement.xlsx`: A spreadsheet with the structured P&L.
- `variance_analysis.xlsx`: A spreadsheet showing variances with conditional formatting (if added) or status columns.
