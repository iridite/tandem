# Our Site - Web Project

This is the source code for Our Site e-commerce platform.

## Project Structure

- index.html - Main homepage
- about.html - About page
- products.html - Products listing (to be added)
- styles.css - Main stylesheet
- app.js - Main JavaScript application
- config.js - Configuration file

## Features

- Product listing with filtering
- Search functionality
- Shopping cart (UI only)
- Responsive design

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Setup

Simply open index.html in a web browser. No build process required.

## Notes

- This is a static site for demonstration purposes
- Some features are mocked or incomplete
- API calls go to test endpoints
