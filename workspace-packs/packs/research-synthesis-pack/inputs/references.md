## Framework References

### NIST Cybersecurity Framework

The NIST CSF provides a common language for understanding cybersecurity risk:

- **Identify**: Asset management, governance, risk assessment
- **Protect**: Access control, data security, maintenance
- **Detect**: Anomalies and events, continuous monitoring
- **Respond**: Response planning, communications, analysis
- **Recover**: Recovery planning, improvements, communications

_Reference_: NIST Cybersecurity Framework 2.0 (2024)

### CIS Controls

The CIS Controls provide prioritized set of actions:

- **Implementation Group 1**: Essential cyber hygiene
- **Implementation Group 2**: Enhanced security for sensitive data
- **Implementation Group 3**: Advanced security for critical assets

_Reference_: CIS Controls v8 (2021)

### ISO 27001/27002

International standards for information security management:

- **27001**: ISMS requirements and certification
- **27002**: Security controls and guidance

_Reference_: ISO/IEC 27001:2022, 27002:2022

### SOC 2 Trust Services Criteria

Security, availability, processing integrity, confidentiality, privacy:

- **Security**: Protection against unauthorized access
- **Availability**: System operational and accessible
- **Processing Integrity**: System processing complete and accurate
- **Confidentiality**: Information designated as confidential
- **Privacy**: Personal information collection and use

_Reference_: AICPA SOC 2 Trust Services Criteria
